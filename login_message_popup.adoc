= Configuring a Login Message
:hardbreaks:
:nofooter:
:icons: font
:linkattrs:
:imagesdir: ./media/

[.lead]
You can configure a message to display as a pop-up for all users when they log in. You choose to display only for the next login for all users, or for all subsequent logins.
